QQ安装

习惯使用QQ的人才会想在Linux下安装QQ（80/90后），安装起来也不是一般的遭罪…
首先说明的是Ubuntu下装的QQ与win下的相比真的很差劲，准备下手的同学们请做好心理准备。

  • 1.下载wineQQ国际版，优麒麟提供下载
  • 2.下载ibpng12-0_1.2.54-1ubuntu1_i386.deb 链接: https://pan.baidu.com/s/1cllr10 密码: 3xqu（这一步是因为ubuntu16.10之后源里面不再有libpng12这个包了，会包依赖错误，需要我们手动安装）

  • 3.终端输入

  • 5.解压wineQQ压缩包，进入该目录后终端输入sudo dpkg -i fonts-wqy-microhei_0.2.0-beta-2_all.deb ttf-wqy-microhei_0.2.0-beta-2_all.deb wine-qqintl_0.1.3-2_i386.deb

  • 6.如果报依赖错误，输入：sudo apt-get install -f
  • 7.解决依赖后再执行步骤5
